\# Burrow Detection in Fields



Проект для распознавания нор (жилых нор грызунов) на фотографиях или спутниковых снимках полей. Использует компьютерное зрение (OpenCV) и преобразование Хафа для поиска круглых объектов.



\## Как это работает



1\. Изображение преобразуется в оттенки серого.

2\. Применяется медианное размытие для подавления шумов.

3\. Метод Hough Circle Transform находит все окружности заданного радиуса.

4\. Найденные круги обводятся зелёным, центры отмечаются красным.
